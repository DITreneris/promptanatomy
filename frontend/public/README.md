# Public assets

Static files in this folder are copied to the build output root.

- **og-image.png** – Social sharing image (Open Graph). Add a 1200×630 px image; used as `og:image` in index.html. Replace with your brand image when ready.
- **llms.txt** – Optional AI-readable index with the site's main public URLs and a short product summary; copied to the build output root.
